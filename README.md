# 极致动态 Dynamicest

本模组十分轻量，主要功能有：

1. 实时检测属性、社交、特质和金钱动态提醒；
2. 给PC的状态条（如疼痛、性奋等）加上动画；
3. 将原版移动端折叠后状态下方用于装饰的点线条，修改为真正的状态条；

### 本模组的目的是让玩家能够更加直观地看到数值的变化。

------

## 【框架】不排斥

本模组兼容性极强，安装了框架（如简易框架或秋枫白桦框架等）或者和其他知名模组共存也不会报错。

------

## 【兼容】绝大部分「模组」新增的属性和社交！

我们使用动态读取技术，实时检测社交、属性数值。因此，只要模组的代码写法合法合规，我们都能够检测到并且在数值变化时动态提醒。
